# Scrapy Web-Scraping Projects

## About
Web-scrapers for short-term rental data implemented with Scrapy. This project deploys 3 web-scrapers onto Airbnb's website, each searching on different criteria, and a web-scraper on AirDNA's website (a short-term rental data aggregation service).

This project is not longer maintained and may or may not be operational.

## Built With
* [Scrapy](https://github.com/scrapy/scrapy)
